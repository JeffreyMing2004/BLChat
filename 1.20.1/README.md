# Bilibili Chat Mod 正式版 (1.20.1)

此目录用于存放正式版发布代码。

## 正式版发布原则

1. **配置简化**：正式版用户界面**仅允许**出现“身份码 (Room Code)”输入框。
2. **安全隔离**：`ACCESS_KEY` 和 `ACCESS_SECRET` 不允许出现在用户配置文件中。
3. **代码迁移**：在未得到明确指令前，禁止将当前根目录下 `src` 中的开发中代码迁移至此目录。
